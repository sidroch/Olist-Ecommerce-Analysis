# 📊 Olist Ecommerce Analysis  
**Tech Challenge – Relatório Executivo de E‑Commerce (Pós Tech FIAP 2026)**

Este projeto tem como objetivo analisar o desempenho do e‑commerce Olist, identificando padrões, oportunidades de melhoria e insights estratégicos para apoiar decisões de negócio.  
A análise combina técnicas de **Data Analytics**, **Storytelling**, **Power BI** e **documentação executiva**.

---

## 🚀 Objetivos do Projeto

- Construir uma visão executiva clara sobre o desempenho do e‑commerce.
- Identificar gargalos logísticos, operacionais e de atendimento.
- Criar indicadores estratégicos (KPIs) para tomada de decisão.
- Desenvolver um dashboard interativo no Power BI.
- Documentar metodologia, insights e recomendações.
