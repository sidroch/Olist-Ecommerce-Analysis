
# 📦 Olist E-Commerce Analysis

## Tech Challenge - FIAP | Data Analytics

---

Este repositório contém a análise exploratória, indicadores, gráficos e conclusões executivas sobre o desempenho comercial e logístico da Olist, com base no dataset público da empresa.

## Estrutura do Repositório

Olist-Ecommerce-Analysis/

- data/                     # Arquivos CSV utilizados na análise
- notebooks/                # Notebooks Jupyter
- graficos_relatorio_olist.ipynb
- README.md                 # Documentação principal


---

🎯 Objetivo do Projeto

O objetivo deste projeto é analisar o desempenho da operação de e‑commerce da Olist, identificando:

1) padrões logísticos
2) comportamento de receita por estado
3) categorias de maior destaque
4) riscos e oportunidades regionais
5) insights estratégicos para tomada de decisão

---

📊 Indicadores e Gráficos Produzidos
1. Percentual de Atrasos por Estado
- Ordenação dos estados pelo percentual de atrasos.
- Destaque para regiões críticas e eficientes.

2. Destaques por Categoria
- Ranking qualitativo baseado em volume, receita e valor agregado.

3. Receita por Estado
- Comparação entre estados com maior potencial de monetização.

4. Riscos e Oportunidades
- Classificação dos estados entre Risco e Oportunidade.

5. RJ vs Mediana — Percentual de Atrasos
- Comparação direta entre o desempenho do RJ e a mediana nacional.
